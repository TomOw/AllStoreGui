About this project

This is my personal project I made during learning Angular.js and improving my knowledge of Spring Boot and Hibernate. It's online store/price comparator. It provides functionality for ordinary users to compare the prices, make orders, write reviews. Store owners/manager can see the orders waiting for further processing, check storage statuses, add/edit/delete items from their offer. Feel free to checkout the whole functionality using sample credentials I provided.
